# QQ Farm Templates

来源仓库：

- `https://gitee.com/luckytiger12138/qq-farm`

用途：

- 作为当前项目 QQ 农场 OCR 流程之外的模板匹配兜底资产。
- 优先覆盖当前最有价值的视觉入口：好友、仓库、商店、商城化肥、出售确认、回家、空地和少量代表性种子。

当前筛选文件：

- 按钮类：`btn_harvest.png` `btn_water.png` `btn_weed.png` `btn_bug.png`
- 好友类：`btn_friend_help.png` `btn_home.png`
- 仓库类：`btn_warehouse.png` `btn_zhongzi.png` `btn_batch_sell.png` `btn_sell.png`
- 商店类：`btn_shop.png` `btn_shop_close.png` `btn_buy_confirm.png` `btn_buy_max.png`
- 通用关闭：`btn_close.png`
- 化肥类：`btn_fertilize_popup.png` `bth_feiliao2_yj.png` `bth_feiliao_pt.png`
- 地块类：`land_empty.png` `land_empty2.png`
- 种子示例：`shop_迎春花.png` `shop_猕猴桃.png` `seed_迎春花.png` `seed_猕猴桃.png`

说明：

- 这不是整仓模板镜像，只保留当前项目最容易直接接入的高价值模板。
- 如果后续要做模板匹配兜底，优先先接：仓库一键出售、商城有机化肥、商店购种、空地识别。
